# Ball and Beam Control System

A complete mechatronics project developed following the VDI 2206 methodology.

## Project Overview

The Ball and Beam system is a classical unstable control problem where the position of a ball is controlled by adjusting the inclination angle of a beam using a servo motor.

The project includes:

- Mechanical Design
- CAD Modeling
- Kinematic Analysis
- Dynamic Modeling
- PID Control Design
- MATLAB/Simulink Simulation
- Arduino Implementation
- System Identification
- FEA Validation
- Experimental Testing

## Hardware Components

- Arduino Uno
- MG995 Servo Motor
- HC-SR04 Ultrasonic Sensor
- Custom PLA Structure
- 12V Power Supply

## Software Tools

- MATLAB
- Simulink
- Arduino IDE
- System Identification Toolbox

## Results

The PID controller successfully stabilized the ball position and achieved good agreement between simulation and experimental implementation.
